1. Format wählen

Alle Maße sind in mm. Beim Drucken bitte 100 % / tatsächliche Größe einstellen.

2. Text & Stil

Erste Zeile = Überschrift. Weitere Zeilen = kleiner Text. Im Listenmodus wird daraus ein neuer Eintrag.
Erweiterten HTML-Modus anzeigen
Nur bei Bedarf. Normalerweise reichen Überschrift und Text darunter.

3. Bild / Logo

Bild auf Etiketten anzeigen
Bild wird auf jedem Label gleich platziert. Für Logos am besten PNG verwenden.

4. Rückseite / Duplex

Rückseite zusätzlich erstellen
Rückseite ist aus. Aktiviere die Checkbox, um Rückseiten in die Druckvorschau einzufügen.
Für normalen Duplexdruck ist die erste Option richtig. Wichtig: Die sichtbare Reihenfolge unterscheidet sich erst, sobald mehr als eine Vorderseite entsteht.

5. Daten & Druck

Alles bleibt lokal im Browser. Bilder im JSON können die Datei größer machen.
